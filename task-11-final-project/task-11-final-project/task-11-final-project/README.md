# EcoShop - E-Commerce Product Listing Website

A modern, fast, and fully responsive e-commerce product listing landing page built from scratch using pure semantic HTML5, CSS3 Custom Properties, and Vanilla JavaScript.

## ✨ Features
- **Responsive Navigation**: Adaptive hamburger menu for mobile devices.
- **Dynamic Product Rendering**: Products are structured as JavaScript objects and rendered dynamically into the DOM.
- **Live Category Filtering**: Smooth client-side filtering without reloading the page.
- **Micro-interactions**: Interactive hover states and a functional cart counter indicator.
- **Client-side Form Validation**: Custom error feedback and a success toast state for the newsletter section.

## 🛠️ Technology Stack
- HTML5 (Semantic Elements)
- CSS3 (Flexbox, CSS Grid, Media Queries, Variables)
- Vanilla JavaScript (ES6+ array methods, DOM manipulation)

## 🚀 Setup & Installation
1. Clone or download this repository to your local system.
2. Ensure your directory matches the proper structural breakdown (`assets/css/`, `assets/js/`).
3. Open `index.html` directly in any modern browser (Chrome, Edge, Firefox).
4. *Optional*: Use the **Live Server** extension in VS Code for hot-reloading development.
